🎬 Movie Search App
📌 Overview
The Movie Search App is a simple web application that allows users to search and explore movies using real-time data from an external API.

It fetches popular movies and displays details such as title, rating, and overview in a clean and user-friendly interface.

🚀 Features
🔍 Search movies by name
🎬 Display popular movies on load
⭐ Show movie ratings with color indicators
📄 View movie overview/details
⚡ Fast and dynamic UI using JavaScript
🛠️ Tech Stack
HTML
CSS
JavaScript
TMDB API (The Movie Database)
⚙️ How It Works
Fetches movie data from TMDB API
Uses JavaScript fetch() for API calls
Dynamically updates UI using DOM manipulation
Handles user input for searching movies
